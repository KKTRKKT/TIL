구글 클래스룸페이스북트위터이메일
HTML과 CSS 기본사항을 배우게 된 것을 축하합니다! 이제 인터넷에서 볼 수 있는 웹 페이지에 관련된 많은 사항을 이해하고, 그러한 웹 페이지를 잘 살펴볼 수 있을 겁니다. 사용되고 있는 HTML 태그와 CSS 속성을 이해할 수 있을 것입니다.
그러나 아직 HTML과 CSS을 기반으로 구축하기 위해 할 수 있는 많은 사항이 있습니다. 계속해서 읽겠습니다!

HTML 태그 더 알아보기

  • 폼 태그(Form tags): <button>과 <input>과 같은 웹 형태를 생성하기 위해 사용되는 일부 태그들이 있습니다. 폼 태그는 일반적으로 서버로 데이터를 다시 전송하는 데 사용됩니다. 해당 태그들은 칸 아카데미에서 활성화시킬 수 없습니다. 또한 폼 태그는 게임을 만들기 위해 JS와 함께 사용될 수 있습니다. 이 부분은 HTML/JS 과정에서 학습할 수 있습니다. 폼 태그에 대해 좀 더 많은 사항은 본 슬라이드를 통해 더 배울 수 있습니다. PHP, Python, Ruby와 같은 서버쪽 언어는 Codecademy에서 배울 수 있습니다.
  • Iframe: <iframe> 태그는 웹 페이지 안의 또 다른 웹 페이지의 틀을 잡아주기 위해 사용되고 유튜브 동영상, 구글 맵스 등을 끼어 넣을 때 유용합니다. 이 태그는 현재 보안과 조정을 위해 Khan 아카데미에서는 비활성화되어 있으나 어디에서나 실험해볼 수 있습니다. 본 슬라이드를 통해 좀 더 배울 수 있습니다.
  • 멀티미디어 태그(Multimedia tags): <audio>와 <video> 태그는 웹 페이지에서 오디오와 비디오 파일을 재생하는 방법으로 현재의 브라우저에서 지원됩니다. <iframe> 태그와 마찬가지로 멀티미디어 태그는 Khan 아카데미에서는 비활성화되어 있으나 어디에서나 이용될 수 있습니다. 본 슬라이드를 통해 좀 더 배울 수 있습니다.
  • 새로운 시맨틱 태그(New semantic tags): 몇년 전에 "HTML5" 사양의 일부로 개발자들을 위해 <article>, <aside>, <nav> 등과 같이 <div> 대신에 사용할 수 있는 새로운 시맨틱 태그가 브라우저에 존재하게 되었습니다. 시맨틱 태그를 이용하려고 할 때 결정하기 위한 유용한 플로우 차트를 포함하여 여기에 새로운 시맨틱 태그와 관련된 내용이 많습니다.

그 외 CSS 선택자(selectors) 와 속성(properties)

칸아카데미에서 다음의 모든 것을 실험해 볼 수 있습니다:
  • CSS3: 몇 년 동안 브라우저에서 지원되어 온 CSS를 포함하지만 "CSS3" 명세서라고 알려진 것에는 새로운 CSS 속성과 선택자가 많습니다. 새로운 버전의 브라우저가 출시될 때마다 계속해서 브라우저는 추가적인 지원을 합니다. 본 슬라이드에서 더 배울 수 있으며 브라우저의 종류에 따라 지원되는 것을 알아보기 위해서는 caniuse.com을 확인하시기 바랍니다.
  • 미디어 쿼리(Media queries): 이것은 동일한 웹 페이지가 더욱 작은 크기로 보여지거나 출력되는 것과 같은 다양한 상황에서 다양한 CSS를 기술할 수 있도록 하는 기술입니다. 본 슬라이드를 통해 좀 더 배울 수 있습니다.
  • CSS 프레임워크(CSS frameworks): 이 프레임워크는 CSS 규칙과 클래스의 집합이며 많은 개발자들은 CSS를 빠르게 작성하기 위해 CSS 프레임워크를 이용합니다. 가장 많이 사용되는 프레임워크는 Twitter Bootstrap입니다. 그러나, ZURB foundation, Pure CSS, Topcoat 등이 더 있습니다.

웹 페이지에서 JS 이용하기

웹 개발자는 JavaScript를 이용하여 사용자의 이벤트에 응답하고 서버에서 동적으로 데이터를 가져오는 인터랙티브한 웹 페이지를 만들 수 있습니다. 이를 위해 HTML 내부에 <script> 태그가 포함됩니다. 해당 태그 내부에는 JavaScript가 들어갑니다. 이후 JS 코드는 브라우저가 모든 웹 페이지에서 이용할 수 있는 함수들의 집합인 "DOM API"를 이용합니다. 개발자는 이를 이용해 쿼리를 하고 해당 페이지를 처리할 수 있습니다.
즉, 사용자와의 소통이 원활한 웹 페이지를 만들고 싶다면 JavaScript(칸아카데미에서 배울 수 있습니다.) 와 DOM AMP의 기본을 둘 다 알아야 합니다. (여기 있는 코스에서 배울 수 있습니다).

칸아카데미 외부에서 웹 페이지 만들기

멋진 웹 페이지를 만들고, 배운 것을 커뮤니티와 공유하는 것도 좋지만 다른 코드 에디팅 환경과 여기에서는 제공하지 못한 기능을 경험하는 것은 멋진 일입니다. 계속해서 브라우저에서 코딩을 하고자 한다면 (특히 크롬북(ChromeBook)을 이용한다면) JSBin.com, JSFiddle.net, Cloud9 IDE 또는 Mozilla Thimble을 사용해 볼 수 있습니다.
브라우저 외부에서 코딩을 하고자 한다면 IDE를 다운로드한 후 웹 페이지를 ".html" 파일로 저장하면 됩니다. 많이 사용하는 IDE로 Notepad++ (Windows only), SublimeText, Adobe Brackets, Coda, TextWrangler가 있습니다.

서버에 웹 페이지 호스팅하기

여기에서 여러분은 만든 웹 페이지를 칸아카데미의 자체 서버에 호스팅합니다. 하지만 자신만의 웹 페이지를 구축하고 싶을 수도 있습니다.
우선 여러분은 사이트를 호스팅할 회사를 찾아야 할 것입니다. 기능과 가격이 다양한 매우 많은 호스팅 제공자가 있습니다. 본 슬라이드에서 보다 많이 배울 수 있습니다.
그 후 "mycoolname.com"와 같은 도메인(domain) 이름을 만들 수 있으며, 그 후 사이트는 여러분 신원의 일부가 됩니다. 이것도 호스팅 제공자에게 맡길 수 있지만 보통 다른 도메인 이름 등록 대행을 활용합니다. 본 슬라이드에서 보다 많이 배울 수 있습니다.
일단 웹 호스팅이 되면 여러분은 FileZilla와 같은 FTP 클라이언트를 이용하여 파일을 서버에 전송할 수 있습니다. 이것은 서버에 직접 연결된 후 서버로부터 파일을 탐색, 다운로드 그리고 업로드할 수 있도록 하는 프로그램입니다. 또한 SSH를 이용하여 서버에 접속한 후 그 서버에 있는 파일을 직접 편집할 수도 있습니다.

지속적인 학습

지속적으로 개선되는 브라우저와 활동적인 웹 개발 커뮤니티 덕분에 HTML과 CSS는 계속 변하고 있습니다. 최신의 정보를 알기 위한 좋은 방법은 기사와 시험용 프로그램들로 링크되는 HTML5 weekly와 같은 뉴스레터에 가입하는 것입니다. 물론 새로운 것을 배울 때마다 어떻게 동작하는지 잘 이해하기 위해 테스트 해봐야 합니다.
